# 工程测算工具功能说明

## 1. 首页

首页提供功能入口，包括坐标转换、坐标量算、地图显示、校验码计算和其他功能。各功能页面返回首页后会保留上一次输入和计算结果。

## 2. 坐标转换

坐标转换支持四类转换：

- 经纬度转 UTM：输入十进制度纬度 B、经度 L，例如 `39.908823`、`116.39747`。北纬东经为正，南纬西经为负。
- UTM 转经纬度：输入北向 X、东向 Y，并选择南/北半球。东向 Y 使用带号编码值，例如 50 带 `534084.68` 输入为 `50534084.68`。
- 度转度分秒：输入十进制度数字，例如 `116.39747`，输出为 `116°23′50.892″` 格式。
- 度分秒转度：输入原数字形式，例如 `116.2350892` 表示 `116°23′50.892″`。

计算结果区域支持长按选择文本并复制。

## 3. 坐标量算与指南

坐标量算用于两点之间的测算，输入 A 点和 B 点的 X、Y、H 值，可选择 6000 密位或 6400 密位。

计算结果包括：

- 水平距离
- 直线距离
- 方位角
- 方位密位
- 高低角
- 高低密位

表盘区域显示实时手机指向，并在表盘上标出目标方位。表盘右下角仅显示方位和高低，完整结果显示在下方结果区。指南功能精度与手机传感器有关，使用前建议先用手机自带指南针校准。

## 4. 地图显示

地图显示调用高德地图接口，支持：

- 显示当前位置
- 经纬度或 UTM 输入目的地
- 地图选点填入当前输入格式
- 在地图上显示目的地 marker
- 点击定位按钮回到当前位置
- 点击“打开高德地图导航”跳转到手机高德地图，由高德地图规划路径并导航

地图页面默认使用 UTM 输入格式。UTM 输入包括带号、北向 X、东向 Y 和半球选择。

## 5. 校验码计算

校验码计算支持 XOR8 和 SUM8。输入十六进制数据后可计算对应校验码，结果支持长按选择文本并复制。

## 6. 其他功能

其他功能页面预留历史记录、参数管理和设备数据解析等扩展入口。
